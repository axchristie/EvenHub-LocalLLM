Local LLM — Privacy Policy

Last updated: May 2026

Local LLM is an Even Hub app for the Even Realities G2 smart glasses. This policy explains what data the app accesses, where it goes, and what the developer does with it.

Summary

The developer of Local LLM collects no data from users. The app does not communicate with any server operated by the developer. All data stays between your device and your own self-hosted infrastructure.

Microphone access

Local LLM requests access to the G2 microphone to capture voice input. Recorded audio is used exclusively to:

  • Transcribe speech to text via a Whisper speech-to-text model running on the user's own server
  • Route voice commands to the appropriate menu option via a language model running on the user's own server
  • Send voice queries to a language model running on the user's own server

Audio is sent directly from your device to your self-hosted open-webui instance over your private Tailscale network. Audio is never sent to any server operated by the app developer, and is never stored by the app itself.

Network access

Local LLM sends HTTPS requests to endpoints configured by the user in the app settings. These endpoints point exclusively to the user's own self-hosted open-webui instance, accessible via their private Tailscale network. The app does not communicate with any fixed external server, analytics service, advertising network, or any server operated by the developer.

Settings and local storage

The app stores user-configured settings (API endpoints, API key, model names, and menu option configuration) in local storage on the user's device via the Even Hub SDK. This data never leaves the device except as part of the network requests described above.

Chat history

If the user enables the "Save chat" option for a menu option, conversation transcripts and model replies are saved to the user's own open-webui instance. This data is stored on the user's own server and is subject to the user's own data practices. The developer has no access to this data.

Children's privacy

Local LLM is not directed at children under the age of 13 and does not knowingly collect any information from children.

Changes to this policy

If this policy changes materially, the updated version will be posted at this URL with a revised date.

Contact

Questions about this privacy policy can be directed to the developer via the Even Hub app listing or the project's GitHub repository.
